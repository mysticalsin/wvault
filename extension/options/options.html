About WVault Extension

🔐

WVault Browser Extension

Version 1.0.0

WVault Browser Extension provides secure password autofill by connecting to your local WVault desktop application. Your credentials are encrypted and stored only in your local vault - never in the browser or cloud.

📖 Documentation 🐛 Report Issue 💬 Support

How It Works

  1. Extension detects login forms on websites
  2. Securely queries your local WVault desktop app
  3. Displays matching credentials (if any)
  4. Fills credentials only when you choose
