## Features
- **Zero-Knowledge Encryption**: Your database is fully encrypted (AES-256-GCM) at rest.
- **Observatory**: Tamper-proof security audit trail with blockchain-style integrity
- **Brute Force Protection**: 3-attempt detection with real-time alerts
- **Secure Cloud & Drive**: Encrypted file storage with drag & drop support.
- **Games Galaxy**: Store game accounts, 2FA backup codes, and API keys
- **Personalized Dashboard**: Your name and time-based greeting on the home screen.
- **Fresh Install Guarantee**: Uninstalling wipes all data for a clean slate.
- **Custom Storage Location**: Move your vault to any drive (USB/SSD) via Settings.
- **Visual Setup Wizard**: User-friendly first-run experience with your name.
- **Celestial UI**: Constellation-themed glassmorphism design with fluid animations.
- **Password Health Dashboard**: Visual security score with color-coded indicators.
- **TOTP 2FA Support**: Built-in authenticator codes with QR scanning.
- **Breach Monitoring**: Check if your passwords have been compromised.
- **Secure Clipboard**: Auto-clears copied passwords after 30 seconds.
- **Panic Mode**: Ctrl+Shift+L instantly locks and hides the app.

## Installation

1. **Download**: `WVault Setup 5.0.0.exe`
2. **Run Installer**: Double-click and follow the setup wizard
3. **Create Account**: Enter your name and create a strong master password
4. **Important**: 
   - If you lose your Master Password, **data cannot be recovered**
   - Store your password in a safe place
   - WVault is zero-knowledge; we cannot reset your password
